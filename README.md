# Rule
* このゲームは高圧洗浄機を使って、汚れた物を洗うゲームです。<br>
* 制限時間内に洗うことができなければ依頼失敗です。<br>
* 依頼を達成するとお金がもらえます。<br>
* お金を使って道具を購入でき、依頼を効率よく達成することができます。<br>
* 依頼を達成していくと、店のランクが上がり店が豪華になっていきます。<br>
